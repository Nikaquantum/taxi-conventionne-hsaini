# Taxi Conventionné CPAM Île-de-France

Site web professionnel pour taxi VSL conventionné CPAM en Île-de-France avec formulaire de réservation WhatsApp.

## Déploiement GitHub Pages

Le site est prêt à être déployé directement sur GitHub Pages :

1. Aller dans **Settings** > **Pages**
2. Source : **Deploy from a branch**
3. Branch : **main** / **/ (root)**
4. Cliquer sur **Save**

Le site sera accessible à : `https://nikaquantum.github.io/taxi-conventionne-hsaini/`

## Fichiers SEO inclus

- `sitemap.xml` — Plan du site pour les moteurs de recherche
- `robots.txt` — Directives pour les crawlers
- `manifest.json` — Manifest PWA
- `404.html` — Page de redirection SPA
- `.nojekyll` — Désactive Jekyll sur GitHub Pages
- Schema.org JSON-LD (TaxiService + FAQPage) intégré dans `index.html`
- Open Graph + Twitter Cards pour le partage social
